monumentjs CLI

The CLI for monumentjs projects

Download .zip Download .tar.gz View on GitHub

This is the CLI for working with a monumentjs project. This will be expanded a lot later on... but for now it is what it is.

To use it npm install -g generator-monumentjs-cli and then yo monumentjs-cli and you are on the road.

Documentation

Main Generator

To use it: yo monumentjs-cli

This will lay down a base project that looks like this:

app.js
package.json
routes.json
routes
    home.js

The routes.json file contains a base route for the / get request that is matched by the home.js route handler. You can take a look at it to see how you can put together your routes as you develop the application. It also installs the dependencies in the package.json file. (monumentjs and its dependencies dot.js and mime).
